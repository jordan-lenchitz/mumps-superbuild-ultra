# 🚨 ultra fork 🚨

both professionally and recreationally i work with [MUMPS](https://en.wikipedia.org/wiki/MUMPS), a programming language from 1966 implemented as FOSS by [GT.M](https://sourceforge.net/projects/fis-gtm/). this is a fork of [MUMPS](https://mumps-solver.org/doc/userguide_5.8.2.pdf), the sparse direct solver from 1996 written in fortran

these two MUMPSes (1966 and 1996) have nothing to do with each other, enjoy

## faq

what does each MUMPS stand for?

MUMPS (**1966**) stands for "Massachusetts General Hospital Utility Multi-Programming System" and some people call it "M"

MUMPS (**1996**) stands for "Multifrontal Massively Parallel Solver"
